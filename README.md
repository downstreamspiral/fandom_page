# **Tank Archive**

A structured collection of armored vehicles and their specifications.

## **Overview**

Tank Archive is a lightweight web application that presents detailed information about tanks from major countries.

It uses static JSON data to organize and display tank specifications in a clean, structured interface — no backend required.

## **Core Features**

Browse tanks by country

View detailed specifications for each tank

Fast and lightweight (no server/database dependency)

Easily extendable by adding new JSON entries

## **Tech Stack**

Frontend HTML, CSS

Logic JavaScript

Data JSON

Tools Git, VS Code

## **Future Improvements**

Search and filtering system

Tank comparison feature

Better UI/UX (cards, animations)

Expand dataset (IFVs, APCs, artillery)
